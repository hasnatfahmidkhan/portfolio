# My Portfolio

A modern, responsive, and interactive personal portfolio website built with the **MERN** stack ecosystem and advanced animations. This project showcases my skills, projects, and professional journey as a developer.

## ğŸš€ Features

-   **Responsive Design:** Fully responsive layout for all devices (Mobile, Tablet, Desktop) using **Tailwind CSS**.
-   **Advanced Animations:** Smooth page transitions, scroll-triggered effects, and interactive elements powered by **Framer Motion** and **GSAP**.
-   **Dynamic Project Filtering:** Filter projects by category and search by technology stack efficiently.
-   **Project Details:** Dedicated pages for each project with image galleries (using **Swiper**), detailed descriptions, and tech stacks.
-   **DarkMode:** Native dark mode support adapting to system preferences or toggled user choice.
-   **Contact Form:** Functional contact form integrated with **EmailJS**.
-   **Routing:** Seamless client-side navigation using **React Router v7**.

## âš¡ Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your system.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/hasnatfahmidkhan/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open in your browser:**
    Navigate to `http://localhost:5173` to view the app.
